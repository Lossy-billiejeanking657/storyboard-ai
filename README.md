## 🧭 First Time Use

After you open storyboard-ai, start with a small project.

1. Create a new storyboard.
2. Add a title for your scene or project.
3. Insert your first frame.
4. Add a short note for what happens in that shot.
5. Repeat for each scene.
6. Move frames around until the order feels right.
7. Save your project when you stop working.

A good first project is a 3 to 5 frame scene. This gives you a quick way to learn how the app works.

## 🎨 Typical Workflow

A simple storyboard workflow looks like this:

- Write the scene goal
- Add the first shot
- Add the next action
- Mark camera changes if needed
- Add notes for dialogue or timing
- Review the full order
- Export or save the final version

This keeps your story easy to read and easy to share.
